# Ray Tracing

## Overview
Simple Ray Tracing system made in OpenGL. All logic is written in fragment shader: ```shaders/rt.frag```

## Features
- Ray Tracing Spheres (Path Tracing)
- Blinn-Phong Illumination
- Antialiasing (SSAA)
- Moveable Camera
